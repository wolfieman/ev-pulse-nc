# Competitions

Competition entries, presentations, and results for the EV Pulse NC project — project showcases, case competitions, and symposia with judging or placement. (Distinct from `../conferences/`, which tracks academic conference submissions.)

Each competition gets its own subdirectory named `<slug>-<year>/`. Inside each, track the entry materials (outline, deck PDF, figures, speaker notes), the result, and provenance.

> Editable decks (`*.pptx`) are kept locally but **gitignored** per the repo's binary policy — the **PDF export** is the canonical tracked artifact.
