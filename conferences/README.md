# Conferences

Conference submissions, attendance plans, and post-conference materials related to EV Pulse NC.

Each conference gets its own subdirectory named `<conference-slug>-<year>/` (e.g., `seinforms-2026/`). Each exposes a short README plus the public deliverables (abstract, and the submitted paper/slides as PDF once finalized); working materials and correspondence are kept local.
